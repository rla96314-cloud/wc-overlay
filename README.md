# wc-overlay ⚽

2022 카타르 월드컵 SBS 16강 그래픽 스타일의 **vMix 방송 오버레이**.
컨트롤 페이지에서 값을 바꾸면 오버레이에 **즉시 반영**됩니다 (SSE 실시간 동기화).
외부 의존성 없음 — `node server.js` 만으로 실행됩니다.

## 실행

```bash
node server.js
```

- **macOS**: `start.command` 더블클릭
- **Windows**: `start.bat` 더블클릭 (Node.js LTS 설치 필요 — https://nodejs.org, 컨트롤 페이지 자동 오픈)
- 포트 변경: `PORT=9000 node server.js` (Windows: `set PORT=9000 && node server.js`)

> Node.js 17 이상 필요. 별도 `npm install` 불필요 — 외부 패키지가 없습니다.

| 주소 | 용도 |
|------|------|
| `http://localhost:8093/`        | **오버레이** — vMix 브라우저 입력 소스로 추가 (1920×1080) |
| `http://localhost:8093/control` | **컨트롤 페이지** — 브라우저에서 열어 조작 |

## vMix 설정

1. vMix → 입력 추가 → **Web Browser**
2. URL: `http://localhost:8093/`
3. Width `1920`, Height `1080`
4. 오버레이 배경은 투명이라 영상 위에 그대로 합성됩니다.

## 구성 요소 (각각 ON/OFF 토글 · 위치/크기 조절 가능)

| 요소 | 위치 | 내용 |
|------|------|------|
| 스코어보드 | 좌상단 | 대회명·라운드, 양팀 국기/이름/점수, **타이머 시계** |
| 골 알림 | 중앙 | ⚽ 타이머 시간 + "골! 팀명" (득점 시 자동 표시→자동 숨김) |
| 득점 기록 | 우측 | 시간·팀·득점자 목록 |
| 중계진 | 좌하단 | 캐스터 / 해설위원 |
| 뉴스 티커 | 하단 | 라벨 + 흐르는 헤드라인(자동/수동) |

> 진출 배너·다음경기는 제거되었습니다.

## 핵심 기능

### 1. 득점 / 골 알림 (컨트롤 상단)
홈·원정 **＋ / −** 버튼. ＋ 누르면 한 번에:
① 점수 +1  ② 골 알림 표시(**타이머 시간 기준** 분)  ③ 득점 기록 추가  ④ **OSC 전송**.
득점자 칸에 이름을 넣으면 기록에 함께 저장됩니다(알림에는 표시 안 함).

### 2. 경기 타이머
**전반/후반** 따로, `▶ 시작 / ⏸ 정지 / ⟲ 리셋`. 시작을 누르면 스코어보드 시계가 흐르고,
골 알림 시간이 이 시계 기준으로 자동 표기됩니다(후반은 45분+).

### 3. OSC 전송 (다른 PC로)
대상 IP/포트와 홈·원정 주소(`/goal/home`, `/goal/away`)를 설정. ＋ 득점 시 해당 주소로
**OSC 메시지(인자 없음)만** UDP 전송. (vMix/조명/다른 합주 PC 등 트리거용)

### 4. 경기 불러오기 (football-data.org)
무료 토큰 발급 → https://www.football-data.org/client/register
컨트롤에 토큰 입력 → **경기 목록 불러오기** → 경기 선택 → **적용** 하면 팀명·국기(crest)·점수 자동 입력.
**라이브 동기화** ON 시 20초마다 점수·득점자 갱신. (대회코드 기본 `WC` = 월드컵)

### 5. 뉴스 티커
- **자동**: 구글뉴스에서 검색어(기본 `월드컵`) 헤드라인을 5분마다 수집(네이버發 기사 포함). `지금 갱신` 버튼.
- **수동**: 자막 문구 직접 입력.

### 6. 위치 / 크기 편집
컨트롤 맨 아래 **위치/크기** 카드에서 각 요소 X·Y·배율 슬라이더 조절. 또는 **편집모드** ON 후
`오버레이(편집) ↗`를 열어 **마우스 드래그로 이동 · 휠로 크기** 조절(자동 저장).

## 팁

- 국기는 이모지(🇰🇷) 또는 이미지 URL 둘 다 가능. 경기 불러오기로 적용하면 자동.
- 마지막 상태는 `.state.json`에 저장되어 재시작해도 유지(타이머는 멈춘 상태로 복원). "전체 기본값 초기화" 버튼으로 리셋.

## 파일

- `server.js` — HTTP + SSE 서버 (타이머·득점·OSC·뉴스·football-data 프록시)
- `public/overlay.html` — vMix용 오버레이 (드래그 편집 지원)
- `public/control.html` — 컨트롤 페이지
- `.state.json` — 현재 상태 (자동 생성)
